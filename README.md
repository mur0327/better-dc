# BetterDC

**dcinside 게시글 열람 경험을 개선하는 Chrome 확장 프로그램**

## ✨ 주요 기능

### 🖼️ 이미지/영상 필터
- 게시글 내 이미지와 영상에 **블러 필터**를 자동 적용
- **우클릭**으로 필터 해제 (모든 미디어 동시 해제)
- 움짤(GIF) 클릭 시 생성되는 이미지도 자동 감지하여 필터 적용

### ⬅️ ➡️ 이전글/다음글 네비게이션
- **개념글 모드**에서 이전글/다음글 버튼 추가
- 현재 페이지에 이전/다음 글이 없을 경우 **인접 페이지**에서 자동으로 가져옴
- 게시글 목록을 **캐싱**하여 네트워크 실패 시에도 폴백 지원
- 말머리(카테고리) 필터 유지

### 💬 댓글 영역 UI 개선
- 댓글 새로고침 버튼을 댓글 수 위로 이동

### 🔊 동영상 볼륨 자동 설정
- 동영상 기본 볼륨을 **10%**로 자동 설정
- 쿠키를 통한 전역 볼륨 저장

## 📁 프로젝트 구조

```
BetterDCFinal/
├── manifest.json          # 확장 프로그램 설정
├── images/
│   ├── icon-16.png
│   └── icon-128.png
├── scripts/
│   ├── main.js            # 진입점 - 모든 모듈 로드 및 실행
│   ├── filter.js          # 이미지/영상 블러 필터 기능
│   ├── post-navigation.js # 이전글/다음글 네비게이션 기능
│   ├── comment.js         # 댓글 영역 UI 개선
│   ├── volume.js          # 동영상 볼륨 설정
│   └── logger.js          # 커스텀 로깅 유틸리티
└── styles/
    └── style.css          # 블러 필터 및 네비게이션 버튼 스타일
```

## 🚀 설치 방법

1. 이 저장소를 클론하거나 다운로드합니다.
2. Chrome에서 `chrome://extensions`로 이동합니다.
3. 우측 상단의 **개발자 모드**를 활성화합니다.
4. **압축해제된 확장 프로그램을 로드합니다** 클릭 후 프로젝트 폴더를 선택합니다.

## 🔧 지원 페이지

- `https://gall.dcinside.com/mgallery/board/view/*` (마이너 갤러리)
- `https://gall.dcinside.com/board/view/*` (일반 갤러리)

## 📝 사용법

1. dcinside 게시글 페이지에 접속하면 확장 프로그램이 자동으로 작동합니다.
2. 이미지/영상은 블러 처리된 상태로 표시됩니다.
3. 미디어를 **우클릭**하면 모든 필터가 해제됩니다.
4. 개념글 모드에서는 하단에 **이전글/다음글** 버튼이 추가됩니다.

## 🛠️ 기술 스택

- **JavaScript (ES6+)**: 동적 모듈 임포트, async/await
- **Chrome Extension Manifest V3**
- **MutationObserver**: DOM 변경 감지
- **SessionStorage**: 게시글 목록 캐싱
